# Vue SSR构建美团网主页

+ based on nuxt.js && vue.js
+ using redis && mongodb
+ element-UI

## Something to start with

``` bash
# to install packages you should run
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run dev

# build for production and launch server
$ yarn run build
$ yarn start
```
